Proposta do projeto e abordagem técnica

O Objetivo desse repositório é avaliar práticas e estruturas de projeto, simular e decupar libs, workflow e padrões de projeto. Os principais tópicos são:

*acesse o repositório para verificar em código
  • - Inicialização do Angular
  • - Localização com i18n
  • - Possibilidades e práticas de teste com TDD
  • - Geração e utilização de sprites svg
  • - Geração de Rotas em runtime antes do ‘bootstrap’ do Angular
  • - Táticas de Lazy Load
  • - Detecção de dispositivo e estratégias de responsividade com Sass, Javascript e brakpoints
  • - Logger
  • - Mock
  • - Centralização de erros
  • - State management minimilista com objetivo de gerar entendimento sobre os patterns utilizados nas principais libs
  • - Aplicação de animação com animejs
